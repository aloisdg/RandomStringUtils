# RandomStringUtils

ReandomStringUtils is a C# port of [org.apache.commons.lang.RandomStringUtils](https://commons.apache.org/proper/commons-lang/javadocs/api-2.4/org/apache/commons/lang/RandomStringUtils.html).

## Support

| Type   | IsIncluded |
| ------ |:----------:|
| Bool   |     x      |
| Int    |     x      |
| Long   |     X      |
| Float  |     X      |
| Double |     X      |
